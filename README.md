# pengiriman
Jadi disini saya membuat aplikasi untuk pengiriman barang.

Bisa perorangan, bisa perusahaan. jadi disini saya memanfaatkan program dari
android untuk mengirimkan barang secara cepat.

Isinya adalah nama pengirim, tujuan, tanggal pengirim, kriteria pengiriman, dan kriteria barang.

Saya menggunakan 2 edit text, 3 text view, 1 button, 2 radio button, 1 spinner dan 3 checkbox

Disini menggunakan button untuk menampilkan hasil yang diinputkan oleh user.

Jika user menginputkan nama pengirim kurang dari 4 huruf maka akan terjadi eror. huruf minimal 4 yang dimasukan.
